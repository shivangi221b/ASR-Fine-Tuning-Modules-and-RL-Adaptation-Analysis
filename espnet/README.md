# ESPNet — ASR Domain Adaptation

This folder will contain a log for all ESPNet experiments for the shared ASR domain-adaptation study, using the same datasets and evaluation protocol as the NeMo experiments in [`../nemo/`](../nemo/).

See the [root README](../README.md) for overall goals and cross-framework progress.

## Repo

Since ESPNet is a big repository, changes to the code are made in a [fork of the repository](https://github.com/GAInTheHouse/espnet).

## Datasets

Same three datasets as NeMo — see [`../data/README.md`](../data/README.md) for details.
